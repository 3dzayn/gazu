## Available data

Many kind of data can be queried through Gazu. Here is the list of all 
available data:

* Assets (constituants of a shot scene)
* Asset types
* Shots
* Sequences 
* Episodes
* Projects
* Persons
* Tasks
* Task comments
* Task status
* Task types
* Time spents
* Comments
* Preview files
* Working files
* Output files
* Output types
* Software
* File status
* Events
* Asset instances
* Notifications
* Search filters
