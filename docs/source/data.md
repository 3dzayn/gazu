## Available data

Here is the list of all available data you can access through the Python client:

* Assets (constituants of a shot scene)
* Asset instances
* Asset types
* Comments
* Episodes
* Events
* File status
* Metadata
* Notifications
* Output files
* Output types
* Persons
* Playlists
* Preview files
* Projects
* Search filters
* Sequences 
* Shots
* Software
* Subscriptions to notifications
* Tasks
* Task comments
* Task status
* Task types
* Time spents
* Working files
